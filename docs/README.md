## 기능 요구사항

- [ ] 2개의 좌표 정보의 정상적인 입력 받기
    - [ ] 좌표 정보 파싱하기
- [ ] 2개의 좌표 정보의 비정상적인 입력 받기
    - [ ] (숫자,숫자)-(숫자,숫자) 의 형식이 아닌경우
    - [ ] 좌표값이 0미만, 24 이상인 경우
    - [ ] 에러 문구를 출력하고 다시 입력 받기
- [ ] 2차원적으로 2개의 좌표 출력하기
- [ ] 두 좌표간의 거리 출력하기

## 프로그래밍 요구사항

- [ ] 자바 컨벤션 지키기
- [ ] 들여쓰기는 4 spaces
- [ ] 들여쓰기의 깊이는 최대 1
- [ ] 3항 연산자 사용하지 않기
- [ ] else 사용하지 않기
- [ ] 단위 테스트를 사용하여 TDD로 구현하기
- [ ] 함수의 라인 길이는 최대 10
- [ ] 모든 원시 값과 문자열을 포장하기
- [ ] 일급 컬렉션을 사용하기
- [ ] 변수명 축약하지 않기
- [ ] 커밋 컨벤션을 지키고, 의미있는 커밋로그 작성하기
