## 기능 요구사항

- [ ] 3개의 좌표의 정상적인 입력값 받기
- [ ] 3개의 좌표 입력값 파싱하기
- [ ] 3개의 좌표의 비 정상적인 입력값 받기
  - [ ] (숫자,숫자)-(숫자,숫자)-(숫자,숫자) 의 형식이 아닌경우
  - [ ] 에러 문구를 출력하고 다시 입력 받기
- [ ] 2차원적으로 3개의 좌표 출력하기
- [ ] 삼각형의 넓이를 출력하기

## 프로그래밍 요구사항

- [ ] 자바 컨벤션 지키기
- [ ] 들여쓰기는 4 spaces
- [ ] 들여쓰기의 깊이는 최대 2
- [ ] 3항 연산자 사용하지 않기
- [ ] else 사용하지 않기
- [ ] 단위 테스트를 사용하여 TDD로 구현하기
- [ ] 함수의 라인 길이는 최대 10
- [ ] 모든 원시 값과 문자열을 포장하기
- [ ] 일급 컬렉션을 사용하기
- [ ] 변수명 축약하지 않기
- [ ] 커밋 컨벤션을 지키고, 의미있는 커밋로그 작성하기

## 학습 내용

### String.replaceAll(String regex, String replacement)

- regex : 변경하고자 하는 대상이 될 문자열의 정규 표현식
- replacement : 대체될 문자열

### String.format(String format, Object arg)

- 인자 arg를 형식 format에 맞춘 문자열 반환
